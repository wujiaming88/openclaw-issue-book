# 问题分类体系

## 分类目录

| 分类 | 目录 | 说明 |
|------|------|------|
| 🔌 Gateway | `issues/gateway/` | 网关启动、绑定、远程连接、端口、SSL/TLS |
| 📡 Channel | `issues/channel/` | Telegram、Discord、WhatsApp、Signal 等频道配置和连接 |
| 🤖 Provider | `issues/provider/` | AI 提供商配置（OpenAI、Anthropic、Bedrock、Google 等）、模型调用、API Key |
| 🧠 Agent | `issues/agent/` | Agent 配置、会话、子代理、工作区、SOUL/IDENTITY/MEMORY |
| 📱 Node | `issues/node/` | 移动端/桌面端配对、连接、Tailscale、QR 码 |
| 🔧 Plugin | `issues/plugin/` | 插件加载、配置、兼容性 |
| 🛠️ Tool | `issues/tool/` | 内置工具、MCP、技能系统、ClawHub |
| 💻 CLI | `issues/cli/` | 命令行使用、openclaw 命令报错 |
| 📦 Install | `issues/install/` | 安装、升级、依赖、环境问题 |
| 🔒 Security | `issues/security/` | 权限、认证、安全配置 |
| ⚡ Performance | `issues/performance/` | 性能、内存、CPU、响应延迟 |
| ❓ Other | `issues/other/` | 无法归入上述分类的问题 |
