# OpenClaw Issue Book 📖

OpenClaw 问题诊断记录手册 — 由 ClawDoctor 自动维护。

## 结构

```
issues/
├── YYYY-MM-DD-简短描述.md    ← 每个问题一个文件
└── ...
```

## 问题模板

每条记录包含：
- **问题描述**：用户遇到了什么
- **环境信息**：相关版本、配置
- **根因分析**：为什么会出错
- **解决方案**：具体修复步骤
- **参考资料**：文档链接、GitHub Issue 等
- **标签**：分类标签便于检索
